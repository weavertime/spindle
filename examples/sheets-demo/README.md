# Pagent Sheets Demo

Standalone demo website for testing Spindle Sheets library functionality.

## Features

This demo showcases:
- Sparse cell storage
- Virtual scrolling for performance
- Cell editing (click, double-click, keyboard)
- Keyboard navigation
- Sample data pre-loaded

## Setup

```bash
cd examples/demo
npm install
npm run dev
```

The demo will run on `http://localhost:5175`

## Usage

1. **Edit Cells**: Click on any cell to edit, or double-click for inline editing
2. **Navigate**: Use arrow keys, Tab, or Enter to move between cells
3. **Scroll**: Scroll around to see virtual scrolling in action
4. **Test Performance**: Try entering data in many cells to test performance
